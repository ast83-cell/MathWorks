Q1: What is an Element?
Sol: An element is a pure substance consisting of only one type of atom

Q2: What is a Compound?
Sol: A compund is a pure substance which is made up of molecules of same kind.

Q3: What is a mixture?
Sol: when two or more substances mix with each other without participating in a chemical change, the resulting substance is called a Mixture.

Q4: Respectively Write the chemical symbol for the following elements:
(i)Hydrogen, (ii)Chlorine, (iii)Calcium, (iv)Oxygen ,(v)Iron
Sol: (i)H, (ii)Cl, (iii)Ca, (iv)O, (v)Fe

Q5: What is a Chemical formula?
Sol: The chemical formula of a substance is based on the composition of that substance. It indicates the type and number of atoms of each kind.

Q6: Write the names of the following symbols:
(i)N, (ii)C, (iii)Cu, (iv)Ag, (v)Au
Sol: (i)Nitrogen, (ii)Carbon, (iii)Copper, (iv)Silver, (v)Gold

Q7: What are ions?
Sol: Many substances are made up of positively and negatively charged particles called ions.

Q8: Write the following ions?
(i)Sodium, (ii)Aluminium, (iii)Hydroxide, (iv)Sulphate, (v)Nitrate, (vi)Acetate
Sol: (i)Na+, (ii)Al3+, (iii)OH-, (iv)(SO4)2-, (v)(NO3)-, (vi)CH3COO-

Q9:Write the chemical formula for the following:
(i)Sodium Carbonate, (ii)Pottasium Nitrate, (iii)Magnesium Oxide
Sol: (i)Na₂CO₃, (ii)KNO3, (iii)MgO

Q10: What is a chemical equation?
Sol: The representation of a reaction using symbols and formulae of substances involved in a reaction is called a chemical equation.

Q11: What is a combination reaction.Give one example?
Sol: A chemical reaction in which two or more substances(reactants) combine to form a 1 new substance called product is a called a combination reaction
eg: Mg + O2 → 2MgO

Q12: What is a decomposition reaction.Give one example?
Sol: A decomposition reaction can be defined as a chemical reaction in which one reactant breaks down into two or more products.
H2CO3 → H2O + CO2

Q13: What is a displacement reaction. Give one example?
Sol: displacement reaction is the one wherein the atom or a set of atoms is displaced by another atom in a molecule
Cu(s) + 2AgNo3(aq) → 2Ag(s) + Cu(NO3)2(aq)
